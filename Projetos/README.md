# Disciplina Projeto
## Módulo 1 - Revisão Sobre Levantamento de Requisitos e Metodologias Ágeis

- O papel do Analista de Requisitas (PO)
    - Intermedias a "Comunicação" entre StakeHolders e a Equipe de Desenvolvimento, Gerenciando o BackLog do Project

- Metodologias de Desenvolvimento de Projetos
    - Clássica / Cascata (Waterfall)
        - Foco no Projeto
        - Uma etapa após a outra
    -  Metodologias Ágeis (Scrum, Kanbam, XP, Lean)
        - 4 Pilares:
            - Foco no Usuário (indivíduos).
            - Entregas Rápidas (Software em funcionamento vale mais que Documentação).
            - Mudanças são bem vindas.
            - Indivíduos e Interações mais que Processos e Ferramentas.
